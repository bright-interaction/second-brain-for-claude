# Example gotcha

A gotcha note records a trap that cost time, so it never costs time again. One trap
per note.

**Do:** write the gotcha the instant you escape it, with the symptom you'd search for.

**Don't:** assume you'll remember it, or leave it as a code comment only.

**Why:** gotchas are tier-0 in Mesh (boosted + budget-reserved), and the next agent
searching the symptom will find your one-liner before re-deriving the fix. Delete this
example once you have a real one.
